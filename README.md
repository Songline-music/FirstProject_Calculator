# FirstProject_Calculator
A desktop calculator simulation built with Qt 6.10.2 and C++.

## Features
- Supports scientific calculator functions (arithmetic operations, parentheses, etc.)
- Fun prompt easter eggs for an interactive user experience
- Always-on-top window mode for convenient multi-tasking
- One-key calculation with Enter key, intuitive operation
- Robust input validation and error handling

## Build from Source
- **Tech Stack**: C++, Qt 6.10.2, CMake
- **IDE**: Visual Studio 2026
- **Compiler**: MSVC 2022 64-bit
- Steps:
  1.  Clone the repository
  2.  Open `CMakeLists.txt` in Visual Studio 2026
  3.  Configure Qt 6.10.2 environment and build the project

## Project Structure
- Calculator/
- ├── Core/ Core calculation logic and expression parser
- ├── Game/ Easter egg and interactive game-related features
- ├── Helper/ Utility classes and helper functions
- ├── Input/ Input handling modules
- ├── Log/ Error throwing and exception handling
- ├── Main/ Program entry and main window implementation
- ├── Output/ Result output and display modules
- ├── Resources/ UI resources, icons and assets
- └── UI/ Qt UI design and interface implementation
